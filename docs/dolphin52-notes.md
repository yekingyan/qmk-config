### ADC 引脚注意

GP26、GP27、GP28、GP29 用作按键输入（SWITCH5/6/11/12），与 Sweep 项目遇到的 ghost keys 问题同源——需关闭 ADC 外设，确保这些引脚作为普通 GPIO 使用。

## 键位设计

参考 `zmk-config/config_silakka54/lily58.keymap`（54 键"双核驱动"过渡版）。

### 设计原则

- **内核**：Sweep 36 键的完整 7 层（Base/Nav/Num/Sym/Mouse/Fun/Media），Callum OSM + Vim Nav
- **外围**：多出来的键全部使用传统键盘布局，方便新手直接上手

### 布局分区（每侧 26 键）

```
┌─────┬─────┬─────┬─────┬─────┬─────┐
│ 外围 │ 外围 │ 外围 │ 外围 │ 外围 │ 外围 │  ← 数字行（传统）
├─────┼─────┼─────┼─────┼─────┼─────┤
│ 外围 │核心 │核心 │核心 │核心 │核心 │  ← Q 行
├─────┼─────┼─────┼─────┼─────┼─────┤
│ 外围 │核心 │核心 │核心 │核心 │核心 │  ← A 行
├─────┼─────┼─────┼─────┼─────┼─────┤
│ 外围 │核心 │核心 │核心 │核心 │核心 │  ← Z 行
└─────┴─────┴─────┼─────┼─────┼─────┘
                  │核心 │核心 │        ← 拇指键
                  └─────┴─────┘
```

| 区域 | 键数（每侧） | 内容 |
|------|-------------|------|
| 核心 | 17 键 (3×5+2) | Sweep 完整键位，7 层不变 |
| 外围 | 9 键 (数字行6 + 修饰列3) | 传统键位：数字/`~/Tab/Shift/Ctrl |

### 外围键默认映射（Base 层）

左手外围：
- 第1行：`` ` `` `1` `2` `3` `4` `5`
- 第2行第1列：`Tab`
- 第3行第1列：`LShift`
- 第4行第1列：`LCtrl`

右手外围（镜像）：

- 第4行第6列：`RShift`

## 双拇指切层（tri-layer）

参考 lily58 设计：内侧两拇指键同时按住时进入「第三个拇指键」对应的层。

- 左 `Space(_NAV)` + `Tab(_NUM)` 同按 → `_FUN`
- 右 `Enter(_SYM)` + `Bspc(_MOUSE)` 同按 → `_MEDIA`

实现：`keymap.c` 的 `layer_state_set_user` 中 `update_tri_layer_state`。
该函数是 vial-qmk 核心功能，无需 `TRI_LAYER_ENABLE`。

## 刷机（进 bootloader）

- `_FUN` 层右手数字行最外侧键 = `QK_BOOT`。
- 刷机和弦：按住左 `Space+Tab`（进 Fun 层）→ 点右手右上角键 → 当前 USB 侧进 bootloader。
- 备用：`config.h` 启用 `RP2040_BOOTLOADER_DOUBLE_TAP_RESET`，双击物理 RESET 也可进 boot。

### 何时需要刷两边？

- **只改键位 / 宏 / combo / 层**：只刷主手（插 USB 那半）即可。从手仅负责扫描矩阵并通过串口发送原始按键事件，keymap 只在主手运行。
- **改了通信协议、矩阵定义、或 QMK 大版本升级**：两边都要刷，确保串口通信协议一致。
- 首次刷机或长时间未更新：建议两边都刷同一份固件。

## 备注

- 52 键分体键盘（左右各 26 键 + SWITCH0 用于 TRRS）
- 布局：4 行 × 6 列 + 2 拇指键（每侧）
- 主控：YD-RP2040
- 连接方式：Direct Pin（非矩阵扫描）
- 触发逻辑：Active Low + Internal Pull-Up
